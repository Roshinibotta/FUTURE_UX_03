# FutureCRM – CRM Dashboard UI/UX Design

##  Project Overview

FutureCRM is a modern web-based CRM (Customer Relationship Management) dashboard designed for small agencies, startups, freelancers, and SaaS businesses to manage leads, clients, tasks, and sales pipelines efficiently.

This project was created as part of the **Future Interns UI/UX Design Internship Task 3 (2026)**.


# Objective

The objective of this project is to create a clean, scalable, and professional CRM dashboard UI that helps businesses:

* Track leads and sales stages
* Manage clients
* Organize tasks and follow-ups
* Monitor sales performance
* Improve workflow efficiency


# 🖥️ Pages Included

## 1️⃣ Login Page

A professional authentication page for existing users.

### Features

* Email & password login
* Remember me option
* Social login buttons
* Modern SaaS UI layout


## 2️⃣ Signup Page

A clean onboarding page for new users.

### Features

* Create account form
* Password confirmation
* Business-style onboarding design


## 3️⃣ Dashboard

The main analytics and business overview page.

### Features

* KPI cards
* Revenue overview chart
* Leads by source graph
* Recent activity section
* Upcoming tasks


## 4️⃣ Leads Page

A structured page for tracking and managing leads.

### Features

* Lead table layout
* Filters & sorting
* Lead score tracking
* Search functionality
* Lead status management



## 5️⃣ Clients Page

A card-based client management interface.

### Features

* Client profile cards
* Revenue tracking
* Renewal dates
* Active / inactive status
* Search & filters



## 6️⃣ Tasks Page

A Kanban-style task management board.

### Features

* Planned tasks
* Upcoming tasks
* Completed tasks
* Overdue tasks
* Task cards with deadlines



## 7️⃣ Pipelines Page

A visual sales pipeline management page.

### Features

* Sales pipeline stages
* Funnel conversion graph
* Revenue trend graph
* Deal tracking
* Pipeline analytics

# 🧩 UI/UX Decisions

* Consistent sidebar navigation across all pages
* Reusable card components for scalability
* Minimal and modern SaaS-inspired layout
* Different layouts used across pages to avoid repetitive design
* Dashboard analytics added for better business insights
* Kanban workflow improves task organization
* Funnel analytics improve pipeline visibility



# 🔄 Workflow Structure

```text
Login / Signup
      ↓
Dashboard
      ↓
Leads → Pipelines → Clients → Tasks
```

The workflow was designed to simulate real CRM software used in SaaS and digital agencies.
